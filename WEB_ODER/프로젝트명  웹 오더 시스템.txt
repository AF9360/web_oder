프로젝트명 : 웹 오더 시스템
개요
가게에서 사용하는 웹페이지 기반 메뉴 주문 서비스를 구현한다
고객은 웹페이지에 접속하여 메뉴를 확인하고 주문을 완료할 수 있으며
관리자는 실시간으로 주문을 확인하고 처리할 수 있다

[고객용 페이지]
-삼품 메뉴 보기
-장바구니 담기 / 수량 변경
-주문하기 (테이블 번호 또는 이름 입력)
-가격 필터 기능

[점주용 페이지]
-주문 실시간 확인
-메뉴 가격 합계 확인
-주문 상태 변경(접수 -> 조리중 -> 완료)
-재고 관리 기능
- 메뉴 관리 기능

3. 기술 및 스택
백엔드 : python + Flask
프론트엔드 : JS + HTML + CSS
DB : SQLite
실시간 연동 : Flask-SocketIO

4.페이지 구성
[고객 화면]
- /menu ; 메뉴 목록
- /cart : 장바구니
- /checkout : 주문 정보 입력
- /complete : 주문 완료 안내

[점주 화면]
- /admin/orders : 주문 현황 리스트
-/admin/oders/<id> :주문 상세 및 상태 변경
-/admin/menu : 메뉴 관리 및 재고 관리

5. 기능 흐름도
I. 고객이 메뉴 페이지에 접속
II. 메뉴 고르고 장바구니에 담기
III. 테이블 번호 및 이름 입력 후 주문 요청
IV. 점주는 주문 알림 받기 및 관리자 페이지에서 주문 확인
V. 주문 상태 변경
